# Brand Guide — Watt Security

## 1. Brand Personality

If Watt Security were a person, they would be:

- A senior operations officer, not a salesperson — calm, exact, and matter-of-fact.
- Someone who has seen every scenario before and is never caught off guard.
- Formal but not cold; approachable in conversation, uncompromising in execution.

**Personality traits:** disciplined, vigilant, composed, dependable, exacting, understated.

**What the brand is NOT:** flashy, aggressive, gimmicky, overly friendly/casual, trend-chasing.

## 2. Core Values

1. **Vigilance** — constant attention, nothing overlooked.
2. **Integrity** — doing the job right whether or not anyone is watching.
3. **Discipline** — process and standards applied consistently, every shift, every site.
4. **Respect** — for clients, the public, and the people Watt Security employs and protects.
5. **Excellence** — continuous training and improvement, not a static standard.

## 3. Desired Image

Watt Security should be perceived as:

- A **premium, professional** security provider comparable in presentation to major international security groups — not a small informal guarding outfit.
- **Locally rooted, internationally credible** — deep knowledge of Douala/Cameroon operating conditions, delivered with international operating standards.
- **The disciplined choice** — the company serious organizations (corporate offices, industrial sites, high-profile events) choose specifically because they cannot afford unreliable security.

## 4. Provisional Tagline

**"Discipline. Vigilance. Protection."**

Rationale:
- Three words, each a core value, in a cadence common to serious institutional and defense-adjacent branding — communicates rigor at a glance, even before reading further copy.
- Works identically in English and French, which matters for a bilingual Cameroonian audience (Douala) without requiring a translated variant.
- Avoids generic security-industry clichés ("Your Safety, Our Priority") in favor of a sharper, more confident statement.

**Secondary/supporting line (optional, for hero subtext):**
"Professional security services for corporate, residential, industrial, and event environments — available 24/7 in Douala and beyond."

> This tagline is provisional and should be validated by the client before being locked into visual assets (logo lockups, letterhead, etc.).

## 5. Tone of Communication

| Context | Tone |
|---|---|
| Website copy | Confident, precise, factual. Short, declarative sentences. No hype adjectives ("amazing," "incredible"). |
| Service descriptions | Specific and operational — describe what is actually done, not vague reassurance. |
| Calls to action | Direct and low-friction: "Request a Security Assessment," "Speak to Our Team," "Get a Quote." |
| Trust-building content | Evidence-led: certifications, experience, coverage, response capability — not adjectives alone. |
| Tone to avoid | Casual/playful language, exclamation marks, emoji, marketing-speak superlatives ("best-in-class," "world-class") without substantiation. |

**Voice checklist for any copy written for the site:**
- Would this sentence sound credible coming from a security operations director? If not, rewrite it.
- Does it state a fact/capability, or just an adjective? Prefer the fact.
- Is it as short as it can be while staying clear? Cut anything that doesn't add meaning.

## 6. Naming & Terminology Consistency

- Refer to the company consistently as **"Watt Security"** (not "Watt," "WS," or "Watt Security Ltd." unless that is the confirmed legal name — to be confirmed with the client).
- Use consistent service names across the whole site, matching the list defined in `README.md` (Corporate Security, Residential Security, Industrial Site Surveillance, Event Security, VIP Protection, Security Guards, Patrol Services, K9 Security Dogs, 24/7 Security Services).

## 7. Open Question for Client Validation

- Confirm the tagline "Discipline. Vigilance. Protection." — or provide an alternative direction if preferred.
- Confirm the exact legal/commercial name to display in the footer and legal pages (e.g., "Watt Security," "Watt Security Sarl," etc.).
